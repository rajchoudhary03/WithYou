# WithYou Project — Agent Rules

## Auto-Execute Rule
**For all changes to the WithYou website (`e:\WithYou`), execute immediately without asking for approval or plan confirmation.**

- Do NOT create an implementation plan and wait for "Proceed"
- Do NOT ask "Should I go ahead?"
- Do NOT request feedback before coding
- Just read the request, build it, and deliver

This applies to: new features, UI changes, bug fixes, refactors, translations, component additions — everything.

Only pause and ask if the request is genuinely ambiguous (e.g., two contradictory design options where a choice is truly needed).
